
계정 내 다중 사용자 분리 전략 보고서

    1. 문제 정의
        - 한 계정에서 여러 사용자가 이용하는 경우, 시청 기록이 섞여 있어 개인화 추천의 정확도가 낮아질 수 있음.
        - 가족 계정, 친구 계정 등에서 각 사용자별 관심사를 분리해야 함.

    2. 데이터 분리 방법

        2.1 프로파일 기반 분리
            - 프로파일 기능 활용:
                - 계정 내 프로파일이 제공되면 각 프로파일별 데이터를 개별 사용자 데이터로 간주.
                - 프로파일 선택 시 자동으로 별도의 데이터베이스에 기록 저장.
            - 장점: 정확한 사용자 구분 가능.
            - 단점: 프로파일 기능을 사용하지 않는 사용자에 대해 한계가 있음.

        2.2 시간대 및 패턴 분석
            - 시청 시간대 기반 분리:
                - 동일 계정에서 발생한 시청 기록의 시간대를 분석하여 사용자 패턴을 분리.
                - 예: 부모가 주로 저녁에 시청, 자녀가 주로 오후에 시청.
            - 콘텐츠 유형 분석:
                - 콘텐츠의 장르, 등급, 주제를 분석하여 사용자 그룹화.
                - 예: 성인 취향(드라마, 액션) vs. 어린이 취향(애니메이션, 가족 영화).
        
        2.3 디바이스 기반 분리
            - 디바이스 구분:
                - 동일 계정 내에서 사용된 디바이스별 시청 기록을 구분.
                - TV, 모바일, 태블릿 등 각 디바이스에 대해 개별 사용자 데이터로 처리.
            - 장점: 동일 계정 내에서도 디바이스가 사용자 특성을 어느 정도 반영.
            - 단점: 하나의 디바이스를 여러 사람이 사용하는 경우 한계.

    3. 데이터 분리 알고리즘

        3.1 클러스터링 기반 사용자 구분
            - K-Means, DBSCAN:
                - 시청 기록 데이터를 클러스터링하여 계정 내 사용자 군집 추출.
                - 주요 데이터: 시청 시간대, 콘텐츠 장르, 평점, 재생 빈도.
                - 클러스터 개수를 자동으로 결정하려면 Silhouette Score 등 사용.

        3.2 개인화 점수 매트릭스 생성
            - 계정 내 각 사용자가 시청했을 확률을 점수화.
            - Collaborative Filtering 또는 Neural Collaborative Filtering(NCF)을 활용.
            - 콘텐츠-사용자 매트릭스에서 계정 내부의 유사 사용자 군집 생성.

        3.3 다중 사용자 분류 모델
            - 시청 기록 기반 분류:
                - 특정 콘텐츠와 사용자 행동 데이터를 결합해 다중 사용자 패턴 학습.
                - 예: 특정 사용자는 한 장르에만 집중, 다른 사용자는 다양한 장르 시청.

    4. 사용자 구분 후 추천 전략

        4.1 동적 사용자 구분
            - 사용자가 로그인 없이 콘텐츠를 선택하면, 선택 기록을 실시간 분석하여 가장 유사한 사용자 프로파일에 매칭.
            - 디바이스 및 시간대 정보 활용.

        4.2 프로파일 추천 강화
            - 프로파일 선택을 장려하기 위한 인터페이스 개선.
            - 예: 프로파일 선택 시 추천 결과 미리보기 제공.

        4.3 가족형 계정 추천 시스템
            - 계정 단위로 사용자 간 교차 추천:
                - 동일 계정 내 사용자의 관심사가 겹치는 콘텐츠를 추가 추천.
                - 예: "가족이 함께 즐길 수 있는 콘텐츠".

    5. 기대 효과
        1) 정확한 개인화:
            - 계정 내 다중 사용자 패턴을 효과적으로 분리해 개인화 추천 품질 향상.
        2) 사용자 경험 개선:
            - 가족형 계정을 사용하는 사용자도 정확한 추천 제공.
        3) 데이터 활용도 증가:
            - 시청 기록의 세분화를 통해 추천 모델의 데이터 품질 향상.
